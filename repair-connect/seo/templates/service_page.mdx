# Service Page Template

This template provides the structure for creating new service category pages.

## File Location
```
/src/app/services/[service-slug]/page.tsx
```

## Page Sections

### 1. Hero Section
- Service badge with icon
- H1: "Professional [Service Name] in Dubai"
- Value proposition paragraph
- Primary CTA: "Get Instant Quotes"
- Secondary CTA: "Find Workshops Near You"
- Trust indicators (3 checkmarks)

### 2. Common Problems Section
- H2: "Common [Service Category] Issues We Fix"
- Grid of 8 problem cards with checkmarks
- Hover effects on cards

### 3. How It Works Section
- H2: "How It Works"
- 3-step process cards
- Numbered icons (1, 2, 3)

### 4. Brands Section
- H2: "All Brands Welcome"
- Badge grid of supported brands
- Hover effects on badges

### 5. FAQ Section
- Uses FAQSection component
- Includes FAQ schema markup
- Accordion-style Q&A

### 6. CTA Section
- H2: "Ready to Fix Your Car?"
- Final value proposition
- Primary CTA button
- Gradient background card

## Metadata Configuration
```typescript
export const metadata = generateServiceMetadata(serviceData);
```

## Schema Markup
```typescript
const serviceSchema = generateServiceSchema(serviceData);
const breadcrumbSchema = generateBreadcrumbSchema([
  { name: 'Services', url: '/services' },
  { name: serviceData.serviceName, url: `/services/${serviceData.serviceSlug}` },
]);
```

## Styling Guidelines
- Use Twitter color theme for primary elements
- Maintain consistent spacing (py-16 for sections)
- Border-2 for prominent cards
- Shadow-lg for elevated elements
- Hover effects on interactive elements

## SEO Checklist
- [ ] Unique, descriptive title (50-60 chars)
- [ ] Compelling meta description (150-160 chars)
- [ ] 4-6 high-priority keywords included
- [ ] H1 contains target keyword + "Dubai"
- [ ] Service schema markup implemented
- [ ] FAQ schema markup included
- [ ] Breadcrumb navigation present
- [ ] Internal links to related pages
- [ ] Mobile-responsive design
- [ ] Accessibility compliance
- [ ] Core Web Vitals optimized
